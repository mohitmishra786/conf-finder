# Confab

A modern, open-source directory for technical conferences.

## Features

- **Conference Discovery**: Browse and filter events by category (AI, Web, DevOps, etc.)
- **CFP Tracking**: Monitor Call for Papers deadlines with urgency indicators
- **Financial Aid**: Identify events offering speaker support or scholarships
- **Zero Cost**: Built on static data with daily automated updates
- **Dark Mode**: Designed for developer ergonomics

## Future Enhancements

- Email subscriptions for new CFPs
- Calendar integration
- Community submissions portal
